# 🎓 EduLine - Course Marketplace

A full-stack online course marketplace where users can explore courses, enroll, and track learning progress.

---

## 🚀 Features

- 🔐 User Authentication (JWT-based login/signup)
- 📚 Browse & Search Courses
- 🛒 Enroll in Courses
- 📊 Dashboard to track enrolled courses
- 🌙 Modern UI with responsive design
- 🎨 Clean split-layout landing page

---

## 🛠 Tech Stack

**Frontend:**

- React (Vite)
- Tailwind CSS

**Backend:**

- Node.js
- Express.js

**Database:**

- MongoDB (Atlas)
